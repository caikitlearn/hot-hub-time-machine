About Hot Hub Time Machine

Hot Hub Time Machine is a data product made to help users plan trips using Hubway. Based on the user's start and end points, the Hot Hub Time Machine will:

  1. Identify the Hubway station closest to your start point, and the Hubway station closest to your end point.
  2. Output walking directions to your start station, biking directions from start to end station, and walking directions from your end station to end point.
  3. Predict, based on your arrival time to a station, the number of bikes at that station (click the bike icon).
  4. Provide two alternative start and end stations, each with predicted number of bikes (click the bike icons).
  5. Update the page as the user changes start and end points.

Hot Hub Time Machine is made by Tai Cai for his Insight Data Science project.
